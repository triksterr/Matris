# Идеи для дальнейшего развитие проекта

## Пушка  

В верхней строке появляется пушка, из которой можно выстрелить числом - возможно не один раз.  
Если попасть в кубик, с которым данное число образует пару для Правила в Раунде - кубик исчезнет.  
Если нет - либо образуется новый кубик, либо изменится число в самом кубике  

## Гравитация

"осыпание" вниз висящих (без опоры снизу) кубиков - "гравитация" - зависит от правил раунда
(Внесено в текущие задачи проекта)
Сделать разные правила - с гравитацией и без!

## Другие:

Функции поведения кубика, меняющие по тактам цвет, координаты и т.п.
По дефолту - заглушка, которая ничего не делает

И точно также - функции поведения фигуры
