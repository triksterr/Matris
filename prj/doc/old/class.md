# Классы

## Кубик
// Cube - один числовой кубик с цветом
 - число  
 - позиция в фигуре или в общей матрице (в координатах стакана)
 - виден/не виден  
 
## Фигура
// Figure - фигура из кубиков
- id кубиков - указатели на кубики
- матрица смещений
- координаты центра (в координатах стакана)

## Слои внизу - без падающей фигуры
// Layers - матрица слоев внизу
- матрица всего стакана - id кубиков или указатели
Обновляется в начале каждого Раунда

## Раунд
// Round - Раунд
- размеры стакана
- скорость падения
- разрядность чисел
- правило
- количество фигур
- правило уничтожения
- минимальное количество кубиков в фигуре
- максимальное количество кубиков в фигуре
- цветовые правила
и т.п.

Игра - цикл из последовательных раундов. В каждом раунде генерируются все правила.

## Функционал ввода-вывода (графика, звуки, клавиатура, файлы)

### IRender - Интерфейс отрисовки, графика (например, методы DrawRect, DrawText).
Реализации:
- CRender - консольный
- WRender - Windows графика
и т.п.

###  IAudio - Интерфейс воспроизведения звуков.
Реализации:
- CAudio - звуки консоли Windows
- WAudio - звуки Windows Audio

###  IInput - Интерфейс обработки ввода (касания, клавиатура, мышь).
Реализации:
- WInput - клавиатура Windows
- MInput - ввод мышью
и т.п.

###  IFiles - Интерфейс работы с файлами (загрузка уровней, сохранение прогресса).
Реализации:
- WFiles - файловый ввод-вывод Windows
и т.п.
