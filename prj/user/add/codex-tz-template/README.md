# Codex TZ Template

Шаблон и стандарт подготовки технического задания для работы в связке GPT + Codex.

Репозиторий содержит один основной файл:

- `AI_TZ_Guide_for_Codex.md` — подробный гайд, который выступает «проводником» между разработчиком, ChatGPT и Codex. Его можно загружать в чат с ИИ и использовать как стандарт внутри команды.

## Зачем это нужно

Идея простая: вместо того чтобы каждый раз «из головы» придумывать промпты для ИИ, мы один раз описываем, как должно выглядеть хорошее ТЗ, и превращаем этот документ в интерфейс между человеком и моделью.

Файл помогает:

- разработчику — шаг за шагом собрать информацию о задаче и оформить её в виде нормального ТЗ;
- ChatGPT — вести интервью и собирать структурированное ТЗ из ответов;
- Codex — получать чёткое, самодостаточное задание, по которому можно уверенно писать и менять код.

## Как пользоваться

Базовый сценарий такой:

1. Откройте этот репозиторий и скачайте файл `AI_TZ_Guide_for_Codex.md`.
2. Создайте новый чат с GPT (или другим ассистентом) и загрузите туда этот файл как «документ со знаниями».
3. В одном-двух предложениях опишите задачу, по которой хотите подготовить ТЗ.
4. Дальше следуйте вопросам модели: она будет вести вас по разделам стандарта и собирать ТЗ.
5. В конце попросите ИИ «собрать итоговое ТЗ по шаблону из файла» и пробежитесь по чек-листу в конце документа.
6. Получившееся ТЗ можно отдать Codex (или живому разработчику) как вход для реализации.
