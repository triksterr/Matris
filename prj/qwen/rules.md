# Project Rules

- Всегда отвечай на русском.
- Язык программирования C++17.

## Code Style
- Пиши чистый код
- Не ломай существующую архитектуру
- Не меняй публичные API без необходимости
- Добавляй комментарии там, где нужно

## Backend
- Используй dependency injection
- Избегай giant classes
- Не дублируй код

## Refactoring
- Делай минимальные изменения
- Сначала анализируй существующий код
- Не переписывай рабочие части без причины и согласования

## Ответы
- Кратко объясняй изменения
- Перед большими изменениями сначала предложи план
