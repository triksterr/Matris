Числовой тетрис
Стакан - все как в тетрисе
Появляется фигура из квадратиков и начинает падать. В каждом квадратике число (число или знак операции)

Для начала это - один квадратик, потом 2, потом три в линию, потом уголком, можно и 4

Нужно ее повернуть и положить так, чтобы получилось верное равенство/неравенство.
Или чтобы при заданной операции и результате получился этот результат по прямым и диагоналям...

Только числа.
В каждом раунде задача, например, сумма двух = 11 

тогда [1][0][6] нужно поставить вертикально на 5

```|[2][5][7][3]|```

На числа и операции:
```
|            |
|   [3]      |
|   [+]      |
|   [1]      |
|            |
|[2][=][7][3]|
|[-][4][+][+]|
|[1][0][=][2]|
--------------
```
